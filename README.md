# DCM for Choco

[DCM](https://dcm.dev/) pkg for [choco](https://community.chocolatey.org/packages/dcm/1.0.0#install).

Windows users can use these pkgs to easily install and update DCM executable.

## Installing

To install the executable:

```sh
choco install dcm
```

## Updating

Simply run:

```sh
choco upgrade dcm
```
